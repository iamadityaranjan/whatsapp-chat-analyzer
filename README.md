<h2>📌 About the Project</h2>

<p>
  WhatsApp Chat Analyzer is a data-driven tool that processes exported WhatsApp chat files and converts them into meaningful insights.
</p>

<p>
  It helps users understand chat behavior by analyzing message frequency, user activity, word usage, emoji patterns, and time-based trends.
</p>

<h2>⚡ How It Works</h2>

<ol>
  <li>Export WhatsApp chat as <code>.txt</code> file (without media)</li>
  <li>Upload file into the Streamlit application</li>
  <li>System parses messages using regex-based preprocessing</li>
  <li>Data is cleaned and structured into analyzable format</li>
  <li>Analytics and visualizations are generated instantly</li>
</ol>
